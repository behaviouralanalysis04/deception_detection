# 🎭 Deception Detection System

AI-powered system for detecting deception through video and audio analysis.

## Features

- **Real-time Video Analysis**: Live camera feed analysis with facial landmark detection
- **Audio Analysis**: Speech pattern, pitch, and energy analysis
- **Combined Scoring**: 60% video + 40% audio weighted scoring
- **Interactive Dashboard**: Streamlit-based web interface
- **History Tracking**: Store and visualize analysis history
